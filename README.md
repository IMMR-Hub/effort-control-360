# EFFORT Control 360

Demo visual para EFFORT CONSULTORA — sistema de control documental, vencimientos, SIGA, liquidaciones y balances.

## Instalación y ejecución

```bash
npm install
npm run dev
```

Abre http://localhost:5173 en tu navegador.

## Presentación comercial

Esta demo muestra cómo EFFORT podría validar en 14 días una capa de control sobre SIGA, Drive y Calendar, usando solo 5 clientes reales, sin reemplazar sistemas actuales y midiendo ahorro de tiempo, riesgos detectados y vencimientos prevenidos.

## Notas

- Demo 100% local, sin backend ni APIs externas.
- Todos los datos son ficticios.
- Stack: React 18 + Vite + Tailwind CSS + lucide-react.
